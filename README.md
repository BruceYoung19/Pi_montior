# 🖥️ Pi Monitor

The intial point of this project was to use a raspberry pi to host a server. However since the change. I have started to a Lenovo Tiny to see if I make my home server more reproducible and test services.

---

## 📌 Project Goals

* 📈 Track active scripts and server status.
* 🐳 Use **Docker** for containerization.
* 🌐 Host via a local server managed by **Nginx**.

---

## 🧰 Tech Stack

* **Nix Package Manager - for reproducible **

---


## 🚀 Future Plans

* Add real-time dashboard for CPU, memory, and temperature monitoring
* Integrate system logs visualization

--- 

## 🖥️ System Information

- **Device**: Lenovo Tiny M710q  
- **OS**: NixOs  
- **Architecture**: ARM64 (`aarch64`)  
- **Kernel**: Linux 6.x
